adb_remote
==========

Control a android device from your Console


warning: used for *nix only

make sure to execute *adb connect xx.xx.xx.xx:yyyy* first

k or ↑          -- up

j or ↓          -- down

h or ←          -- left

l or →          -- right

space or enter  -- enter

x               -- back

=               -- volume up

\-               -- volume down

m               -- menu
